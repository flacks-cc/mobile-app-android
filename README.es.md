## 🚀 Inicio rápido

### Requisitos previos

- Android Studio Hedgehog (2023.1.1+)
- JDK 21
- Android SDK 34

### Instalación

```bash
git clone https://github.com/flacks-cc/mobile-app-android.git
cd mobile-app-android
./gradlew assembleDebug
```
